# Quicky

A Manifest V3 Chrome extension with a thunder logo. It opens in the browser side panel and lets users manage links that can be inserted into text fields with `s.`.

## Load Locally

1. Open `chrome://extensions`.
2. Turn on Developer mode.
3. Choose **Load unpacked**.
4. Select this folder: `/Users/yagyarajl_500394/Documents/New project 3`.

## Use

- Click the Quicky extension icon to open the side panel.
- Add or edit saved links in the sidebar.
- In any normal input, textarea, or contenteditable editor, type `s.` to open the dropdown.
- Continue typing to filter, such as `s.l`, `s.lin`, or `s.lint`.
- Press Enter to insert the selected link URL.
- Use Arrow Up and Arrow Down to change the selected item.

The first matching item is selected automatically whenever the dropdown opens or the filter changes.
